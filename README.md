## Documentation
### Setup Instructions
1. Clone the repository:  
   ```bash
   git clone https://github.com/dimitrisabra/musote-subscription-hub.git
   ```  
2. Navigate to the directory:  
   ```bash
   cd musote-subscription-hub
   ```  
3. Install dependencies:  
   ```bash
   npm install
   ```  

### Usage
After setting up the project, you can start it using:
```bash
npm start
```

### API Endpoints
- `GET /api/subscriptions` - Fetch all subscriptions
- `POST /api/subscriptions` - Create a new subscription
